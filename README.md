# Avances Territoriales
Esta aplicación fue adaptada por el grupo Semillero de Innovación Geográfica (SIG) de Esri Colombia. Para mayor información consulte www.esri.co o visite @geo_geeks en twitter.
Esta aplicación permite visualizar avances territoriales de diferentes sectores a nivel nacional.

## Instrucciones 

Con el fin de que la aplicación funcione correctamente, es necesario enviar por parámetro en la url:

1. IdSector : Identificador del sector
2. IdVariable : identificador del indicador
3. Año : Periodo 

por ejemplo: 

http://geoapps.esri.co/DNP2/?sector=23&indicador=3348&periodo=2010

Donde sector=[IdSector], indicador=[IdVariable] y periodo=[Año]
